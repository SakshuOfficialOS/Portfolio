Thumbnails Library

A dedicated showcase for every thumbnail. Add new files inside assets/thumbnails/, create a matching entry in assets/thumbnails.json, and the grid updates automatically.

How to add new thumbnails

  1. Add your image file to assets/thumbnails/.
  2. Open assets/thumbnails.json.
  3. Add a new object with the file name, title, category, date, tags, and optional client or link.

Use a local server like VS Code Live Server. fetch() will not load JSON properly from file://.
